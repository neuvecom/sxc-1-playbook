プリセットは**ループ音源**（オレンジ／パープルのパッド）と**ワンショット音源**（イエロー／ブルーのパッド）に色分けして登録されている。ループは [Beat Sync](#システム設定) でマスターテンポに自動同期できる。

## 各部の名称

| 部位 | 役割 |
|------|------|
| OLEDディスプレイ | 波形・設定など各種情報表示 |
| 十字ボタン | メニュー選択・値の変更 |
| バンク選択ボタン A/B/C/D | 演奏時に音源バンクやシーケンスを瞬時に切り替え（各4つまで登録） |
| スライダー（INPUT VOL / MAIN VOL） | 録音入力レベル／再生音量の調整 |
| FX1 / FX2 ボタン・ダイヤル | エフェクトのON/OFF・種類選択・パラメーター変更。EDIT時は波形編集にも使用 |
| 機能ボタン | ▶/■（再生/停止）・REC（録音）・ONE SHOT・LOOP・DEL（削除）・EDIT |
| パッド部（16個） | 音の再生・録音・編集。各パッドに音源を割り当て |

### 接続端子（上面・側面）
- **PHONE 端子**（3.5mm ステレオミニ）：ヘッドホン
- **LINE OUT 端子**（3.5mm ステレオミニ）：ミキサー／外部スピーカー
- **AUDIO IN 端子**（3.5mm ステレオミニ）：外部機器からのサンプリング入力
- **DATA 端子**（USB-C）：スマホとのデータ通信（MIDI／オーディオ転送）
- **POWER/DATA 端子**（USB-C）：電源供給＋データ通信（MIDI／オーディオ転送）
- **電源スイッチ** / **INPUT SELECT スイッチ**（MIC：内蔵マイク / ♪：AUDIO IN / USB）

> ⚠️ PHONE端子にヘッドホンを挿すと内蔵スピーカーからは音が出ない。
> ⚠️ USBケーブルは指定品（付属）のみ使用。

## 用語：バンクとシーケンス

- **バンク** … 16パッドに登録された音のグループ。最大80バンク作成可能。
- **シーケンス** … 1バンクの音で作る最大8小節の演奏パターン。最大50パターン。

A〜Dボタンに登録しておくと、バンクとシーケンスをそれぞれ4つずつ瞬時に呼び出せる。

## 基本操作フロー

```
演奏モード ──A〜D──> バンク選択
   │  │
   │  └─長押し──> シーケンス選択 ──A〜D──> シーケンスモード
   │
   ├─REC ──> RECモード（サンプリング）
   ├─DEL ──> DELモード（音源削除）
   ├─EDIT ──> EDITモード（音程・再生位置編集）
   └─EDIT長押し ──> システム設定（Beat Sync, MIDI Ch など）
```

### 音を鳴らす
1. **A** ボタンでバンク1を選択（十字ボタンでバンク番号変更：上下で±1、左右で±10）
2. オレンジ／パープルのパッドを押してループ音源を再生
3. 合わせてイエロー／ブルーのパッドでワンショットを重ねる
4. **ONE SHOT** と **LOOP** ボタンで鳴り方を切替（両方ONで押すたびループ再生継続）
5. **▶/■** で停止

| ボタン | ON のとき | OFF のとき |
|--------|----------|-----------|
| ONE SHOT | 最後まで再生 | 押している間だけ再生 |
| LOOP | 最後まで再生後、自動で頭に戻り繰り返し | 最後まで再生してそこで停止 |

## サンプリング（録音）

### 内蔵マイクで録音
1. 空のパッドがあるバンク（**BANK 15 以降**）を選択
2. **INPUT SELECT** スイッチを **MIC** に
3. **INPUT VOL** で録音レベル調整
4. **REC** ボタン → サンプリングモードメニュー表示
5. 空のパッド（点滅中）を押す → 録音待機。音を入力すると **AUTO TRIGGER** で自動的に録音開始
6. 完了は選択中パッドを押すか、もう一度 **REC**

### 外部音源を録音
- **AUDIO IN 端子**接続時：INPUT SELECT を **♪**
- **USB 端子**接続時：INPUT SELECT を **USB**
- あとはマイク同様に INPUT VOL 調整 → REC → パッド選択

---

## シーケンサー

1. A/B/C/D のいずれかを**長押し** → 十字ボタンで保存先を選び **A** で決定（最大50パターン）
2. 十字ボタンでカーソル移動、**A** で打ち込み（白く塗りつぶし）、**B** で削除
   - 例：小節1の1拍目でパッド1、2拍目でパッド2…
3. **▶/■** で再生
4. **EDIT** で使用バンクや BPM を変更（再度EDITでシーケンスモードに戻る）

### リアルタイム入力（パッドで打ち込み）
- 再生中に **REC** を押すと、叩いたパッドが再生位置に自動入力される。再生に合わせてタイミングよく叩く。

### 演奏モードからシーケンス再生
- 選択中のバンク選択ボタン（点灯）を長押し → 演奏モードに戻る
- **▶/■** ＋ シーケンスを作ったバンク選択ボタンを同時押しで再生

## エフェクト

再生中の音に **FX1 / FX2** を独立してかけられる（同時使用可）。

| 操作 | 内容 |
|------|------|
| FX1/FX2 ダイヤルを回す | エフェクトの種類を選択 |
| FX1/FX2 ボタンを押す | エフェクト ON/OFF |
| ON中にダイヤルを回す | パラメーター変化 |

**搭載エフェクト**：フィルター / フェイザー / フランジャー / ビットクラッシャー / ロール ×4種 / ディレイ ×2種

---

## 音の編集（EDITモード）

1. **EDIT** ボタンを押す → 編集したいパッドを押す
2. 十字ボタンでパラメーター値を変更

| パラメーター | 内容 |
|-------------|------|
| PITCH | 音程（再生スピード／ピッチ） |
| VOLUME | 音量 |
| GROUP | グループ指定（同グループは同時に鳴らない＝チョーク。ハイハットのオープン／クローズ等に） |
| COLOR | パッドのバックライト色 |

3. **FX1 ダイヤル**で再生開始位置、**FX2 ダイヤル**で終了位置を調整（音源の一部だけ切り取って再生）
4. **EDIT** を押して保存（LED消灯）

---

## システム設定

**EDIT ボタンを長押し**で SYSTEM SETTING に入る。下ボタンでスクロール、左右で値変更、再度 EDIT で抜ける。

| 項目 | 内容 |
|------|------|
| AUTO Trig Lv | オートトリガー録音の感度 |
| Beat Sync | ループのテンポ自動同期 ON/OFF |
| LED Bright | パッドLEDの明るさ |
| Disp Bright | ディスプレイの明るさ |
| MIDI IN Ch. | MIDI受信チャンネル（初期値1） |
| MIDI OUT Ch. | MIDI送信チャンネル（初期値1） |
| APO Time | オートパワーオフ時間（初期20分） |

### 初期化（工場出荷状態に戻す）
EDIT長押し → システム設定 → 十字ボタンで `Initialize` を選び右ボタン → A で実行。
※プリセット音源を削除してしまった場合も初期化で復元可能。サンプリングしたデータは消えるので注意。

## MIDI（USB-MIDI）

ファームウェア Ver.1.1.1 で **MIDI IN/OUT・Control Change の送受信**に対応（USB経由）。
**5ピンDIN端子は無い**ため、PC／スマホとUSB-Cで接続して使う。デフォルトモードはモード3（オムニオフ・ポリ）。

### ノート・マッピング（送信36〜99 / 受信36〜115）
パッド1〜16が Bank A/B/C/D に応じて Note 36〜99 に割り当て。

| Pad | Bank A | Bank B | Bank C | Bank D |
|-----|--------|--------|--------|--------|
| 1 | 36 | 52 | 68 | 84 |
| 2 | 37 | 53 | 69* | 85 |
| … | … | … | … | … |
| 16 | 51 | 67 | 83 | 99 |

※ Note **100〜115**（受信のみ）は、バンクやモードに関わらず「現在のパッド1〜16を押した時」と同じ動作（100→Pad1）。
※ ノートオン・ベロシティは ×（固定）。

### 主なコントロール・チェンジ（CC）
全ボタン・ダイヤルが CC を送受信できるため、DAW から SXC-1 を操作したり、SXC-1 をコントローラーとして使える。

| 機能 | CC No. | 値 | 備考 |
|------|--------|----|------|
| INPUT VOL スライダー | 11 | 0–127 | 連続 |
| MAIN VOL スライダー | 7 | 0–127 | 連続 |
| FX1 ダイヤル | 16 | 0–127 | 右回し+1/左回し-1（送信時）|
| FX2 ダイヤル | 17 | 0–127 | 同上 |
| バンク A/B/C/D | 80/81/82/83 | 押=127, 離=0 | |
| 十字 上/下/右/左 | 85/86/87/89 | 押=127, 離=0 | |
| ▶/■ | 102 | 押=127, 離=0 | |
| REC | 103 | 押=127, 離=0 | |
| ONE SHOT | 104 | 押=127, 離=0 | |
| LOOP | 105 | 押=127, 離=0 | |
| DEL | 106 | 押=127, 離=0 | |
| EDIT | 107 | 押=127, 離=0 | |
| EFFECT FX1 | 108 | 押=127, 離=0 | |
| EFFECT FX2 | 109 | 押=127, 離=0 | |
| INPUT SWITCH | 110 | MIC:0 / AUDIO IN:64 / USB:127 | |

> MIDI Ch は システム設定 の MIDI IN Ch. / MIDI OUT Ch. で変更（電源オフでも保持）。

## 遊び倒すアイデア

- **街の音をビートに**：内蔵マイクで地下鉄の走行音・カフェの食器音・雨の水たまりを録音 → パッドに割り当ててリズム化。単一指向性マイクの収音は良好。
- **名機サウンドで一発ネタ**：MT-40 の “Sleng Teng”、CZ-101、SK-1 を土台にダブ／レゲエ／チップチューン。
- **Beat Sync ループ重ね**：テンポ同期ループを複数パッドに置き、リアルタイムにミュート切替してライブ演奏。
- **チョークグループ活用**：GROUP パラメーターでハイハットのオープン／クローズや、同時に鳴らしたくない音を排他制御。
- **波形トリミング**：EDITモードで FX1/FX2 ダイヤルを使い、長尺サンプルの「美味しい所」だけを抜き出す。
- **DAW連携**：USB-MIDI で DAW に Note/CC を送り、SXC-1 を MIDI コントローラー兼サウンドモジュールとして活用。
- **A〜Dへ即呼び出し**：よく使うバンク／シーケンスを A/B/C/D に登録し、ライブで瞬時に切替。
- **寝転びビートメイク**：電池駆動＋内蔵スピーカーでどこでも制作（連続約2時間が目安、長時間はUSB給電）。

## 実践レシピ（サンプリング遊び）

「ちゃんとした曲を作る」より、**身近な“あの音”をサンプリングして破壊して遊ぶ**のが SXC-1 の楽しみ方。
スマホ／PCの音は **AUDIO IN** か **USB** から、生活音は **内蔵マイク** から取り込める（[サンプリング手順](#サンプリング録音)参照）。

> 💡 鉄則：**「元ネタがギリギリわかるか分からないか」までエフェクトで崩す**のが一番気持ちいい瞬間。

### レシピ1：呼び込み君 × バニラ コラージュ
1. ループ素材（例：呼び込み君のメロディ）を1パッドに長めにサンプリング → **LOOP** ON で背景に流す
2. 別パッドに「バーニラ！バニラ！」を1フレーズ録音
3. **EDIT** → そのパッドで **PITCH** を大きく下げて低音ホラー化／上げて宇宙人化
4. ループに合わせてパッドを連打、または[シーケンサー](#シーケンサー)で配置して自動再生
5. 仕上げに **FX1=ディレイ** をかけてスクラッチ風の残響を足す

### レシピ2：ポテトのティロリ → テクノにビルドアップ
1. マクドナルドの「ティロリ♪」を1パッドにサンプリング、**LOOP** ON
2. [シーケンサー](#シーケンサー)でキック等のワンショットを4つ打ちに打ち込み、土台のビートを作る
3. 盛り上げ局面で **FX1ダイヤル** を回し **フランジャー／ディレイ** を強めにかけ「ティティティ…ロリリリ！」とうねらせる
4. **ロール（×4種）** で高速連打フィルを差し込み、ブレイク前を演出
5. **MAIN VOL** や **フィルター** で抜き差しして緩急をつける

### レシピ3：ファミマ入店音 → Lo-Fi チル
1. 入店音メロディをサンプリング
2. **EDIT** → **PITCH** を少し下げてテンポ感を緩める
3. **FX1=フィルター** で高域を削る → 真夜中のコンビニ風のエモいLo-Fiに
4. **FX2=ビットクラッシャー** を軽くかけてレトロな質感を追加
5. ゆるいループとして流し、上に生活音サンプルを重ねる

### 仕込みのコツ
- **取り込み**：スマホをLINE/USBに繋ぎ、YouTube等で対象音を再生しながら **AUTO TRIGGER** でガシガシ録音
- **音色破壊**：`PITCH`（ピッチ）／`FX`（フィルター・ビットクラッシャー・フランジャー・ディレイ・ロール）／波形トリミング（EDITのFX1/FX2ダイヤルで開始・終了位置）を組み合わせる
- **配置**：似た役割の音は同じ色（`COLOR`）に、同時に鳴らしたくない音は同じ `GROUP` に入れてチョーク
- **保存**：よく使うネタは A〜D ボタンに登録して即呼び出し

> ⚠️ SXC-1 に **アイソレーター／ミキサー／専用ドラムマシンは非搭載**。低音カット等は **フィルター**、リズムは **サンプル＋シーケンサー** で組む。
> ⚠️ 外部入力（AUDIO IN）を“録らずに”リアルタイムでFX通過させられるかは公式に明記なし（[要確認](#既知の制約注意点)）。基本は「サンプリングしてから鳴らす」フロー。
> ⚠️ 公開・配信する場合は、サンプリング元の音源（楽曲・効果音・店舗BGM等）の**著作権・権利処理**に注意。個人で楽しむ範囲に留めるのが無難。

## 既知の制約・注意点

- ⚠️ 連続バッテリー駆動は **約2時間**（eneloop時）。長時間は USB-AC 給電推奨。
- ⚠️ **5ピンDIN MIDI 端子は非搭載**。MIDIはUSB-MIDIのみ（PC/スマホ接続が前提）。
- ⚠️ ノートオンの**ベロシティは固定**（×）。強弱表現はMIDI経由でも不可。
- ⚠️ サンプル合計時間は **約15分**と有限。容量管理が必要。
- ⚠️ オートパワーオフ初期20分。設定でOFFにすると電力を消費し続けるので注意。
- ❓ インポート音声のビート・スライス割り当て可否は要・実機／取説確認。

## トラブルシューティング

| 症状 | 対処 |
|------|------|
| 音が出ない | MAIN VOL を確認 / PHONE・LINE OUT の接続確認（PHONE挿入時は内蔵SP無音）|
| 録音できない | INPUT SELECT スイッチの位置 / ストレージ残量（DELで削除）/ 空パッド選択 / 外部機器の接続 |
| アプリと接続できない | 付属 USB Type-C ケーブルで接続されているか確認 |
| 電源が入らない | POWER/DATA に USB-AC 接続を確認 / 電池交換 |
